# HubSpot vs Pipedrive Comparison Tool

A simple interactive tool for comparing HubSpot and Pipedrive CRM features. Perfect for sales teams to quickly access competitive information when talking to prospects.

## Features

- Interactive feature comparison between HubSpot and Pipedrive
- Question answering system for common customer queries
- Searchable feature database
- Competitive battle cards for sales objection handling

## Demo

You can see a live demo of this tool here: [GitHub Pages Demo Link] (once deployed)

## How to Use

1. Simply open the index.html file in your browser
2. Use the search box to find specific features
3. Click on any feature button to see a detailed comparison
4. Ask questions in the customer question box to get instant answers

## Local Development

To run this project locally:

1. Clone the repository
2. Open index.html in your browser
3. No build steps required!

## Deployment

This project is designed to be deployed directly to GitHub Pages with no build process required.

1. Push your code to GitHub
2. Enable GitHub Pages in your repository settings (Settings > Pages)
3. Select the main branch as the source

## Technologies Used

- React (via CDN)
- Tailwind CSS (via CDN)
- Babel for JSX transformation (via CDN)
